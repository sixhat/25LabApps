HTMX Loading Indicators

( Loading ... )

O atributo hx-indicator do htmx é utilizado para mostrar um indicador visual enquanto um pedio AJAX está a ser executado.

Carregue esta página com as Ferramentas de Desenvolvimento do browser abertas e simule uma ligação lenta no separador Network/Rede para ver melhor o Indicador em funcionamento.

Ver o artigo sobre indicators online em htmx.org/attributes/hx-indicator/

Este exemplo utiliza o HTMX para a parte de interação, o Tailwind para definir os estilos e Daisy UI para utilizar componentes baseados em Tailwind.
